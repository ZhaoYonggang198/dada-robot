# clothing size convertor

## manual
convert_size(category, from_standard, to_standard, size)
example:
convert_size("men shoes", "China", "Italy", "38")
convert_size("men,shoes", "China", "Italy", "38")
convert_size("men;shoes", "China", "Italy", "38")

category is a list of tags with space, comma, semi-colon as delimiters
